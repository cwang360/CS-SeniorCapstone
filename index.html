About

This is a website created from scratch by Cynthia Wang, a Computer Science Major at MOT Charter High School, for her senior capstone project. Here, you will find everything about CS at MOT, from freshman year to senior year. Navigate to each course in the CS major using the nav bar to learn about projects, languages, etc. the students at MOT explore.

This site uses HTML, CSS, and JavaScript. Responsive navigation bar logistics based on W3School's tutorial. Parallax scrolling effect uses Rellax (a vanilla javascript library). Font on this site is Roboto Light from Font Squirrel.
